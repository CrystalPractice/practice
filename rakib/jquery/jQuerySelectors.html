jQuery Selectors

Selectors

In simple words, you can say that selectors are used to select one or more HTML elements using jQuery and once the element is selected then you can perform various operation on that. All jQuery selectors start with a dollor sign and parenthesis e.g. $() It is known as the factory function

The $() factory function

Every jQuery selector start with thiis sign $(). This sign is known as the factory function

Example:

$('p') selects all paragraphs'p'in the document. $('#real-id') selects a specific element in the document that has an ID of real-id. $('real-class') selects all elements in the document that have a class of real-class.

If you have any confliction with theuse of dollor sign $ in any JavaScript library then you can use jQuery() function instead of factory function $(). The factory function $() and the jQuery function is the same.

Different jQuery Selectors

Selector Example Description
